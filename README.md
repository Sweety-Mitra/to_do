A fully functional Task Management Application built with react that supports advanced state handling, undo/redo functionality, and async synchronization.

---

## 🚀 Features

- **Task Management**
  - Create, edit, delete
  - Mark tasks as complete or pending


- **Undo/Redo Functionality**
  - Step backward and forward through task changes
  - Maintains a history stack of user actions

- **Async API**
  - UI updates instantly
  - Simulated API calls usinh asunc functions
  - Rollback on API failures
---

## 🛠️ Tech Stack

- React (functional components + Hooks)
- JavaScript (ES6+)
